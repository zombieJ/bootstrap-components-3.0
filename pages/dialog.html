Dialog

Support Bootstrap style alert box. Call by Javascript without dom init.

Usage

Via JavaScript

Create Dialog box through $.dialog function.

$.dialog({
    title: 'Hello',
    content: 'This is a dialog!',
});

Options

Options should be passed via JavaScript.

Name Type Default Description
title string null Set the dialog title.
content string | html null Set the dialog content.
close boolean true Set dialog box contains close button.
confirm boolean false Set dialog style.true for dialog with close button. false for dialog with Cancel and Confirm button.
buttons array null Customize the buttons on the bottom. Setted will disable the confirm option. e.x:
[{name: "delete", class: "btn btn-danger", left: true}, {name: "Not Yet", value: -1}, {name: "That's Time!"}]
fade boolean true Set dialog fade animation.
size string normal Set dialog size:small,normal,large.
And other options is same in Modal Component.

Events

Event Type Description
callback Trigger on user close the dialog. Calls the value of clicked button.
return false to prevent dialog close. 
$.dialog({
    title: 'Comfirm',
    content: 'Do You want to confirm?',
    confirm: true,
}, function(val){ // val is the value of clicked button.
    // do something...
    return false;// return false will prevent dialog box close.
});
