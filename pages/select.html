Select

Usage

Via data attributes

Create as dropdown. For dropdown-toggle add attr data-type="selector". Append <span data-value> inside. This element will be the container which display selected value. For <ul> list add attr data-type="selector".

<div class="dropdown">
    <button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown" data-type="selector">
        <span data-value></span> <span class="caret"></span>
    </button>
    <ul class="dropdown-menu" role="menu" data-type="selector">
        <li><a>Option 1</a></li>
        <li><a>Option 2</a></li>
    </ul>
</div>

Options

Options need be passed via data attributes. Append the option name to data-, as in data-to="".

Name Type Default Description
to element null Attr of dropdown-toggle. Set target element value when option selected. (Suggest setting to <input> element)

Events

Event Type Description
change Trigger on changing select value. It will also trigger change event for data-to element. 
$('#mySelect').change(function () {
    // do something...
})
